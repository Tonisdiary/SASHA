App Loading Error

The main application failed to load. This could be due to:

  • JavaScript bundling issues
  • React/Expo initialization problems
  • Environment configuration errors
Troubleshooting Steps:
  1. Check the browser console for errors (F12 or right-click > Inspect > Console)
  2. Try clearing your browser cache and reloading
  3. Ensure all required environment variables are set
  4. Restart the development server with the --clear flag
