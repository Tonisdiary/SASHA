Study Buddy Web Debug

This page helps diagnose issues with the Study Buddy web app.

If you're seeing this page: The main app might not be loading correctly. Try the troubleshooting steps below.

Troubleshooting Steps

  1. Check if environment variables are loaded correctly
  2. Clear your browser cache and cookies
  3. Try using a different browser
  4. Check browser console for errors (F12 or right-click > Inspect > Console)
