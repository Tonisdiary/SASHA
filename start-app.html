Start Study Buddy App

Follow these instructions to start the app and use demo mode:

Step 1: Open Command Prompt

Open a command prompt or terminal window on your computer.

Step 2: Navigate to the Study Buddy Directory

Type or copy-paste this command:

cd c:/Users/tonys/Downloads/study-buddy

Step 3: Start the Web App

Type or copy-paste this command:

npm run web

Wait for the app to start. You should see a message that says "Starting project at..." and eventually "Webpack compiled successfully".

Step 4: Open Demo Mode

Once the app is running, open this URL in your browser:

http://localhost:19006/demo-mode.html

Or click the button below after the app is running:

Open Demo Mode

Note: Keep the command prompt window open while using the app. Closing it will stop the app from running.
